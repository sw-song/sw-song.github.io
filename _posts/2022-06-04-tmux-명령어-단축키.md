> 설치

```
$brew install tmux
```

> 세션 시작

```
$tmux new -s session_name
```

> 실행중인 세션 리스트

```
$tmux list
```

> 실행중인 세션 연결

```
$tmux attach -t session_number (or session_name)
```

> 화면 분할(위/아래로)

(ctrl + b) "

> 화면 분할(왼쪽/오른쪽으로)

(ctrl + b) %

> 분할 화면간 이동

(ctrl + b) o

> 세션 유지하고 나가기

(ctrl + b) d
