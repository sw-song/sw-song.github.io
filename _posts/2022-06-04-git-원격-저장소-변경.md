기존 git의 커밋을 유지한 채 원격 저장소 위치만 변경

```
git remote set-url –push origin new_repo_name
```

단, 이 때 새 저장소에 뭔가 커밋된 이력이 있으면 pull 해서 저장소와 맞춘 다음 push 하거나

강제로 덮어써야한다.(force option 사용)

```
git push -f origin main
```
