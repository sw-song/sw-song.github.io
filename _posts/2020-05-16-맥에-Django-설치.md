**% python -m pip install django**

-먼저, 장고를 설치한다.

**% django-admin startproject mysite .**

-데이터베이스와 서버 세팅을 도와준다.

![](/assets/images/posts/40-0.webp)

그럼 장고를 설치한 폴더의 하위에 mysite 폴더가 생성되고, 아래와 같이 settings.py를 확인할 수 있다.

![](/assets/images/posts/40-1.webp)

열어서 몇가지 세팅을 해준다. TIME\_ZONE은 웹에서 시간대 표시할 때 참고하는 지역이니 Asia/Seoul로 설정해주고,

STATIC\_ROOT = os.path.join(BASE\_DIR, 'static') 를 통해 정적 파일 경로를 설정해준다.

![](/assets/images/posts/40-2.webp)

ALLOWED\_HOSTS는 빈값으로 되어 있다. 아래와 같이 채워준다.

![](/assets/images/posts/40-3.webp)

데이터베이스는 sqlite3으로 구동되는 것으로 보인다.

![](/assets/images/posts/40-4.webp)

저장 후 terminal로 돌아가서 migration을 해준다.

![](/assets/images/posts/40-5.webp)

migration이 에러 없이 성공적으로 완료되었다면, 이제 서버를 실행시켜본다.

![](/assets/images/posts/40-6.webp)

http://127.0.0.1:8000/ -> 서버가 생성되었다! 들어가보면 이렇게 장고 페이지가 보인다.

![](/assets/images/posts/40-7.webp)



**관성을 이기는 데이터**
